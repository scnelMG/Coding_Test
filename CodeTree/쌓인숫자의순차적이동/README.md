# 쌓인 숫자의 순차적 이동

- **XP**: 90XP  
- **평균 소요 시간**: 180분  
- **정답률**: 64%  
- **총 제출 수**: 922회  
- **Time Limit**: 3000 ms  
- **Memory Limit**: 176 MiB  

---

## 🧩 문제 설명

N × N 격자판에 **1 이상 N×N 이하의 숫자**가 주어지며,  
각 숫자는 **정확히 한 번씩만 등장**합니다.

이제 총 **M번**에 걸쳐 특정 숫자들을 **순서대로 이동**시키려고 합니다.  
이동할 숫자 번호들이 주어지며, 각 숫자는 아래 규칙에 따라 움직입니다.

---

## ✅ 이동 규칙

어떤 숫자 X를 이동시키려 할 때,

1. 숫자 X가 있는 칸을 기준으로 **8방향(상하좌우 + 대각선)** 으로 인접한 칸들 중  
   **숫자가 존재하는 칸들**을 확인합니다.
2. 그 중 **가장 큰 값**이 적혀있는 숫자가 있는 칸으로 이동합니다.
3. 이동하려는 칸에 이미 다른 숫자(들)가 쌓여 있다면,  
   이동해온 숫자(들)는 그 칸의 **가장 위에 쌓입니다.**
4. 만약 이동 대상 숫자 X의 8방향 인접 칸에 **아무 숫자도 없다면**, X는 **이동하지 않습니다.**

---

## ✅ “쌓여있는 숫자” 이동 규칙

어떤 숫자 위에 다른 숫자들이 쌓여 있는 상태에서 **아래에 있는 숫자**를 이동시키게 되면,

- 그 숫자 위에 쌓여 있던 숫자들도 **함께 이동**합니다.  
  (즉, 선택된 숫자부터 “위로 쌓인 묶음”이 통째로 이동)

---

## 🎯 목표

주어진 M번의 이동을 모두 수행한 뒤,  
각 칸에 쌓인 숫자들을 **위에서부터 순서대로** 출력합니다.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, M**이 공백으로 주어집니다.
2. 두 번째 줄부터 **N개의 줄**에 걸쳐 각 행의 숫자 N개가 공백으로 주어집니다.
   - 1 이상 N×N 이하의 숫자가 **정확히 한 번씩만** 등장합니다.
3. 다음 줄에 이동할 숫자 **M개**가 공백으로 주어집니다.
   - 순서대로 이동을 수행합니다.

---

## 📤 출력

총 **N×N줄**을 출력합니다.  
출력 순서는 다음과 같습니다.

- (1행 1열), (1행 2열), ..., (1행 N열),
- (2행 1열), ..., (N행 N열)

각 칸에 대해,

- 해당 칸에 숫자가 있다면: **가장 위에 적혀있는 숫자부터 아래로** 공백으로 출력
- 해당 칸이 비어있다면: `None` 출력

---

## ⚙️ 제한 조건

- \(2 \le N \le 20\)
- \(1 \le M \le 100\)
- 이동할 숫자: \(1 \le \text{주어지는 수} \le N \times N\)
- 1 이상 N×N 이하의 숫자가 **정확히 한 번씩만** 등장
