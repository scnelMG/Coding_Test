# 도미노 정렬하기

- **정답률**: 33%
- **총 제출 수**: 63회

---

## 🧩 문제 설명

코디는 크기가 **1부터 N까지**이며 **서로 다른** 도미노 블록 N개를 가지고 있습니다.  
원래는 블록이 **오름차순(1, 2, 3, ..., N)** 이 되도록 정렬되어 있었지만,
누군가 도미노의 순서를 섞어버렸습니다.

당신은 도미노를 다시 오름차순으로 되돌려야 합니다.  
단, 도미노 블록의 순서는 아래 방법으로만 바꿀 수 있습니다.

### ✅ 허용되는 이동 방법

- 블록을 하나 선택하여, 그 블록을 **도미노의 가장 앞(front)** 또는 **가장 뒤(back)** 로 이동시킨다.

최소 이동 횟수로 오름차순 정렬을 만들어야 합니다.

---

## 🎯 목표

뒤섞인 도미노 배열이 주어질 때,  
허용된 이동 연산만 사용하여 블록을 오름차순으로 만들기 위한  
**최소 이동 횟수**를 출력합니다.

---

## 📥 입력

1. 첫 번째 줄에 도미노 블록의 수를 나타내는 정수 **N**이 주어집니다.
2. 두 번째 줄에 뒤섞인 도미노 배열을 나타내는 정수 `h1, h2, ..., hN`이 주어집니다.
   - `h_i`는 앞에서 i번째 위치한 블록의 크기를 의미합니다.

---

## 📤 출력

- 블록의 크기가 오름차순이 되도록 하는 **최소 이동 횟수**를 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 10^6\)
- \(1 \le h_i \le N\)
- 모든 도미노 블록의 크기는 서로 다릅니다. (즉, 순열)

---

## ✅ 입력 예제 1

**입력**

```
5
5 1 4 3 2
```

**출력**

```
3
```
