# K개 중에 1개를 N번 뽑기

- **XP**: 40XP  
- **평균 소요 시간**: 19분  
- **정답률**: 78%  
- **총 제출 수**: 5,090회  
- **Time Limit**: 3000 ms  
- **Memory Limit**: 176 MiB  

---

## 🧩 문제 설명

1 이상 K 이하의 숫자 중 하나를 고르는 행위를 **N번 반복**할 때,
나올 수 있는 모든 **서로 다른 순서쌍(길이 N의 수열)** 을 출력하는 프로그램을 작성합니다.

예를 들어 K=3, N=2라면 가능한 결과는 총 9개이며 다음과 같습니다.

```
1 1
1 2
1 3
2 1
2 2
2 3
3 1
3 2
3 3
```

---

## 📥 입력

- 첫 번째 줄에 정수 **K, N**이 공백으로 주어집니다.

---

## 📤 출력

- 가능한 모든 서로 다른 순서쌍을 **한 줄에 하나씩** 출력합니다.
- 각 줄에는 숫자들을 공백으로 구분하여 출력합니다.
- 출력 순서는 **앞에서부터 봤을 때 사전순(lexicographical order)** 으로 앞서는 순서부터 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le K \le 4\)
- \(1 \le N \le 8\)

---

## ✅ 입력 예제 1

**입력**
```
2 2
```

**출력**
```
1 1
1 2
2 1
2 2
```
