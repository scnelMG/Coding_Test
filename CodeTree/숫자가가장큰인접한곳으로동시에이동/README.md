# 숫자가 가장 큰 인접한 곳으로 동시에 이동

- **XP**: 60XP  
- **평균 소요 시간**: 87분  
- **정답률**: 36%  
- **총 제출 수**: 5,635회  

---

## 🧩 문제 설명

1 이상 100 이하의 숫자로 이루어진 **N × N 격자판**이 주어집니다.  
서로 다른 위치에서 시작한 **M개의 구슬**이 있으며, 구슬들은 매 초마다 다음 규칙에 따라 **동시에 이동**합니다.

### ✅ 이동 규칙 (1초에 1번)
각 구슬은 현재 위치에서 **상/하/좌/우 인접한 칸** 중

1. 적힌 숫자 값이 **가장 큰 칸**으로 이동합니다.
2. 만약 “가장 큰 값”을 가진 칸이 여러 개라면, **상하좌우 방향 순서**로 우선순위를 두어  
   가능한 칸 중 **우선순위가 더 높은 칸**으로 이동합니다.
3. 격자 밖으로는 이동할 수 없습니다.

### ✅ 충돌 규칙
구슬들이 이동을 마친 뒤,

- 어떤 칸에 **2개 이상 구슬이 모이면**, 그 칸의 구슬은 **모두 사라집니다.**
- 이동 후 각 구슬의 위치가 모두 다르면, 충돌은 발생하지 않습니다.

---

## 🎯 목표

격자 정보와 초기 구슬 위치가 주어질 때,  
**T초 후 격자판에 남아있는 구슬의 수**를 출력하세요.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, M, T**가 공백으로 주어집니다.
2. 두 번째 줄부터 **N개의 줄**에 걸쳐 격자 정보가 주어집니다.
   - 각 줄에는 N개의 정수가 공백으로 주어집니다.
3. 그 다음 줄부터 **M개의 줄**에 걸쳐 각 구슬의 시작 위치 **r, c**가 공백으로 주어집니다.
   - (r, c)는 r행 c열을 의미합니다.
   - 모든 구슬의 시작 위치는 서로 다릅니다.

---

## 📤 출력

- **T초 이후** 격자판에 남아있는 구슬의 수를 출력합니다.

---

## ⚙️ 제한 조건

- \(2 \le N \le 20\)
- \(1 \le M \le N \times N\)
- \(1 \le T \le 100\)
- \(1 \le r, c \le N\)
