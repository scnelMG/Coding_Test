# 겹치지 않게 선분 고르기

- **XP**: 80XP  
- **평균 소요 시간**: 142분  
- **정답률**: 43%  
- **총 제출 수**: 3,105회  

---

## 🧩 문제 설명

수직선 위에 N개의 선분이 주어집니다.  
각 선분은 구간 \([l_i, r_i]\) 형태로 표현됩니다.

이 중에서 **서로 겹치지 않게** 선택할 수 있는 선분의 **최대 개수**를 구하세요.

⚠️ 주의:  
- 선분이 구간을 **조금이라도 공유하면 겹친 것**으로 봅니다.
- **끝점을 공유하는 경우도 겹친 것**으로 봅니다.  
  (예: [1,2] 와 [2,3] 은 겹친 것으로 처리)

---

## 📥 입력

1. 첫 번째 줄에 선분의 개수를 나타내는 정수 **N**이 주어집니다.
2. 두 번째 줄부터 N개의 줄에 걸쳐 각 선분의 정보 `l_i r_i`가 공백으로 주어집니다.
   - `l_i, r_i`는 i번째 선분의 양 끝점을 의미합니다.

---

## 📤 출력

- 겹치지 않게 뽑을 수 있는 **최대 선분의 수**를 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 15\)
- \(1 \le l_i < r_i \le 1000\)

---

## ✅ 입력 예제 1

```
3
1 2
3 4
5 6
```

## ✅ 출력 예제 1

```
3
```

---

## ✅ 입력 예제 2

```
3
1 2
1 4
3 4
```

## ✅ 출력 예제 2

```
2
```
