# 주사위 던지기

- **XP**: 90XP
- **평균 소요 시간**: 180분
- **정답률**: 64%
- **총 제출 수**: 1,008회

---

## 🧩 문제 설명

6개의 면을 가진 주사위가 1개 주어집니다.

- 각 면에는 1 이상 6 이하의 숫자가 적혀 있습니다.
- **서로 마주보는 두 면의 합은 항상 7**입니다.
- 주사위의 **초기 방향(어떤 면이 위/아래/동/서/남/북인지)** 은 문제의 그림에 주어진 상태로 고정입니다.

이 주사위를 N × N 크기의 빈 격자판 위의 특정 위치에 올려놓고,
총 M번에 걸쳐 상/하/좌/우 중 한 방향으로 **딱 1칸**씩 굴립니다.

### ✅ 격자판에 숫자 쓰기 규칙

- **처음 주사위를 올려놓는 순간도 포함**하여,
- 주사위가 격자판의 어떤 칸에 닿을 때마다  
  해당 칸에는 **주사위의 아랫면 숫자**가 적힙니다.
- 이미 숫자가 적혀 있던 칸이라도 **새 숫자로 덮어씁니다.**

### ✅ 이동 불가 처리

굴리는 도중, 주어진 방향으로 1칸 이동하면 격자 밖으로 나가게 되는 경우에는

- **주사위를 굴리지 않고(상태 변화 없음)**
- 그 다음 과정(다음 명령)으로 넘어갑니다.

---

## 🎯 목표

초기 주사위 위치와 굴리는 방향들이 주어질 때,  
모든 과정을 수행한 뒤 **격자판에 적힌 숫자들의 합**을 출력하세요.

---

## 📥 입력

1. 첫 번째 줄에 정수 **N, M, r, c**가 공백으로 주어집니다.
   - N: 격자 크기 (N×N)
   - M: 주사위를 굴릴 횟수
   - (r, c): 초기 주사위 위치 (r행 c열)
2. 두 번째 줄에 길이 M의 굴리는 방향 정보가 공백으로 주어집니다.
   - 각 문자는 `L`, `R`, `U`, `D` 중 하나
     - `L`: 왼쪽, `R`: 오른쪽, `U`: 위, `D`: 아래

---

## 📤 출력

- 모든 명령을 처리한 뒤, 격자판에 적혀있는 숫자들의 **총합**을 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 100\)
- \(1 \le M \le 10{,}000\)
- \(1 \le r, c \le N\)

---

## ✅ 입력 예제 1

```
4 4 2 3
L D R U
```

## ✅ 출력 예제 1

```
16
```

---

## ✅ 입력 예제 2

```
4 4 2 3
L L L L
```

## ✅ 출력 예제 2

```
11
```
