---

## 🧩 문제 설명

0 이상의 정수 **N**을 2진법으로 나타낸 뒤,  
그 이진수에서 **정확히 한 자리(bit)만 바꿔서** 만든 수를 **a**라고 합니다.

이때, **a의 이진 표현**(0과 1로만 이루어진 문자열)이 입력으로 주어질 때,  
가능한 **N** 중 **최댓값**을 구하는 프로그램을 작성하세요.

---

## 📥 입력

- 첫 번째 줄에 0과 1로만 이루어진 문자열 **a**가 주어집니다.
- a의 맨 앞자리는 0이 아님이 보장됩니다.

---

## 📤 출력

- 조건을 만족하는 가능한 **N** 중 **최댓값**을 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le |a| \le 10\)
- a는 0과 1로만 이루어져 있습니다.
- a의 맨 앞자리는 0이 아닙니다.

---

## ✅ 입력 예제 1

**입력**

```
1010
```

**출력**

```
14
```

### 예제 설명

- 14를 2진법으로 나타내면 `1110`입니다.
- `1110`에서 **두 번째 자리 하나를 바꾸면** `1010`이 됩니다.
- 14보다 더 큰 수에서 한 자리만 바꿔 `1010`을 만드는 방법은 없습니다.
