# 가능한 수열 중 최솟값 구하기

- **XP**: 80XP
- **평균 소요 시간**: 180분
- **정답률**: 48%
- **총 제출 수**: 1,562회
- **Time Limit**: 3000 ms
- **Memory Limit**: 176 MiB

---

## 🧩 문제 설명

길이가 **N**인 수열을 숫자 **4, 5, 6**으로만 구성하려고 합니다.

하지만 다음 조건을 만족하면 그 수열은 **불가능한 수열**로 간주합니다.

- 임의의 길이 **L(≥1)** 에 대해,
- 서로 **인접한 두 연속 부분 수열**이 **완전히 동일**한 경우

즉, 어떤 위치에서든 다음 형태가 존재하면 불가능합니다.

- `... [A][A] ...`
- 여기서 `A`는 길이 L인 연속 부분 수열(문자열)이며, 두 A는 **서로 바로 붙어있어야** 합니다.

문제는 길이가 N인 **가능한 수열**들 중에서,
앞에서부터 읽었을 때 **사전순(lexicographical order)으로 가장 앞서는 수열**을 출력하는 것입니다.

---

## 📥 입력

- 첫 번째 줄에 정수 **N**이 주어집니다.

---

## 📤 출력

- 길이가 N이고, `4, 5, 6`으로만 이루어진 **가능한 수열 중**
  사전순으로 가장 앞선 수열을 출력합니다.
- 숫자 사이에는 **공백 없이** 출력합니다.

---

## ⚙️ 제한 조건

- \(1 \le N \le 80\)

---

## ✅ 입력 예제 1

**입력**

```
2
```

**출력**

```
45
```

---

## ✅ 입력 예제 2

**입력**

```
3
```

**출력**

```
454
```

---

## ✅ 입력 예제 3

**입력**

```
7
```

**출력**

```
4546454
```
